마리오 게임

물체 목록
  1. 플레이어
  2. 몹(잡아야 할 몬스터)
  3. 벽

--플레이어--

  if '오른쪽 화살표키' 누르면:
    플레이어 왼쪽으로 이동

  if '왼쪽 화살표키' 누르면:
    플레이어 오른쪽으로 이동

  if '스페이스바' 누르면:
    플레이어 점프

  if 'R 키' 누르면:
    플레이어가 바라보는 방향으로 공격

--몹--

  실행(지정된 곳에서 좌우로 왔다갔다 하며 움직이기. 무한반복) 

--점수--

  점수 = 60
  1초가 지날 때마다 실행(점수 -= 1)

  if 점수 = 0:
    게임 종료

  // 즉, 점수는 시간과 같다.

--충돌 이벤트 설정--

  if '플레이어' and '몹' 충돌:
    플레이어 => 잠시 색깔이 변하고 1초 후 원상복구(맞았다는 것을 알리기 위하여)
    몹 => 소멸
    점수 => 10 감소

  if '플레이어의 공격' and '몹' 충돌:
    몹 => 소멸
    점수 => 10 증가

  if '플레이어' and '오른쪽 벽' 충돌:
    다음 맵으로 이동

  if '플레이어' and '최종 목적지' 충돌:
    게임 종료
